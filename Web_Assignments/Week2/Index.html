What is the difference between TCP/IP and HTTP

TCP is a transport protocol that transports a file between two nodes.
HTTP is an application protocol that runs over TCP and makes sure it ends up as a complete file at the destination

Define URL:

a URL is the refernece of a web address that gives its location on a network

What is the Client/Server Model?

the distributed application structure that partitions tasks or workloads between the providers of a resource

What two things does the browser do?

  2. allow users to view the information retrieved / display
